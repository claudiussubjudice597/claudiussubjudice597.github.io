## ✨ Main Features

## 🛡️ Data Privacy

Privacy remains the primary goal of this tool. The software uses WebAssembly to run code directly in your browser memory. Data stays on your hard drive. No external server receives your email address, message body, or attachments. You can disconnect your network cable or turn off Wi-Fi while using this tool. It functions identically offline.

## 🔧 Troubleshooting

*   **Application hangs:** Close the browser and restart the application. Large files over 5GB might require more time to index.
*   **Cannot open file:** Confirm that the file is not currently open in Outlook. Outlook locks files while it is running. Close Outlook before you try to open the file here.
*   **Missing images:** Ensure your browser settings allow local file access.
*   **Search returns no results:** Trigger a fresh scan by selecting the "Refresh" icon in the top menu.

## 🖥️ Performance Tips

*   Close other browser tabs if your computer runs slowly.
*   Use a browser version from the last six months for the best results.
*   Store your PST files on a solid-state drive (SSD) for faster loading speeds.
*   Clear your browser cache if the interface seems unresponsive after extended use.
